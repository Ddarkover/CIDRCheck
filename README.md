# CIDRCheck — IP Whitelist Checker

Веб-инструмент для проверки, входит ли IPv4-адрес или домен в whitelist CIDR-диапазонов.

## Ссылки

- **GitHub Pages (демо):** https://ddarkover.github.io/CIDRCheck/
- **Репозиторий:** https://github.com/Ddarkover/CIDRCheck
- **Источник whitelist-данных:** https://github.com/hxehex/russia-mobile-internet-whitelist

## Возможности

- Проверка **IPv4** и доменов на совпадение с whitelist CIDR.
- Разрешение домена через **Cloudflare DNS-over-HTTPS**.
- Быстрый поиск совпадения по CIDR (сортировка + бинарный поиск).
- Вывод IP-информации: ASN, ISP, география, IP, сеть (через `api.ip.sb`).
- Полностью **frontend-only** (без backend), подходит для GitHub Pages.
- Светлая/тёмная тема с поддержкой системной темы устройства.

## Как пользоваться

1. Откройте сайт по ссылке GitHub Pages.
2. Введите IPv4 (`8.8.8.8`) или домен (`domain.tech`).
3. Можно вставлять URL-вид: `https://www.domain.tech/` — поле автоматически нормализуется до `www.domain.tech`.
4. Нажмите **«Проверить»**.
5. Посмотрите вердикт и детальную информацию в карточках ниже.

## Технологии

- `index.html` — структура интерфейса.
- `style.css` — темы, layout, карточки, адаптив.
- `app.js` — загрузка whitelist, резолв доменов, проверка CIDR, вывод результатов.

## Ограничения

- Проверка IP реализована для **IPv4**.
- Геоданные зависят от внешнего API (`ipapi.co`) и его доступности/лимитов.
- Для домена используется A-запись (IPv4).

## Лицензия

Проект распространяется по лицензии, указанной в репозитории.
